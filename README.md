🌐 Portfólio

Projeto desenvolvido para exibir meus trabalhos, habilidades e experiências como desenvolvedor Frontend/Fullstack, com foco em tecnologias modernas e boas práticas de desenvolvimento.

🚀 Objetivo

Treinar e demonstrar:

Criação de interfaces modernas e responsivas

Organização de projetos em React/Next.js

Integração de componentes reutilizáveis

Estruturação de um portfólio pessoal para mostrar projetos e habilidades

🏗 Estrutura do Projeto

components → Componentes reutilizáveis (botões, cards, headers)

pages → Páginas do site (Home, Projetos, Sobre, Contato)

styles → Arquivos de estilização (Tailwind e CSS)

public → Imagens, ícones e recursos públicos

📌 Funcionalidades

Exibição de projetos com descrição, tecnologias e links

Seção de habilidades (skills)

Sobre mim com resumo profissional

Formulário de contato funcional

Navegação responsiva entre seções

🛠 Tecnologias utilizadas

React

Next.js

TypeScript

Tailwind CSS

Vercel (deploy online)
